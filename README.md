# operations on one graph
A Python library of operations on one graph

# Quick start
```
$ git clone https://github.com/Himmelstein/ObeuBonn
$ cd ObeuBonn
graph1 $ make init
```

# Run test
```
ObeuBonn $ make test
```

# Generate documentation
```
ObeuBonn $ ./make_docu
```
Documentation is located at docs/html/
